# ToDo API (Матрица Эйзенхауэра) — v2.0.0

Учебный проект по изучению **FastAPI**, выполненный студентом группы **БСБО-12-22** Арзуманянцем М. Э.

## Описание
Сервис предоставляет API для управления списком задач, классифицированных по методу Эйзенхауэра (важность/срочность). 
**Версия 2.0.0:** Переход на асинхронную работу с базой данных **PostgreSQL (Supabase)** с использованием **SQLAlchemy 2.0**.

## Основные эндпоинты (v2)
Все эндпоинты API имеют префикс `/api/v2`.

* `GET /` — информация об API и версии.
* `GET /health` — проверка статуса API и подключения к базе данных.
* `GET /api/v2/tasks` — получение полного списка задач.
* `GET /api/v2/tasks/{id}` — получение задачи по её идентификатору.
* `GET /api/v2/tasks/stats` — статистика задач.
* `GET /api/v2/tasks/search?q=...` — поиск задач (минимум 2 символа).
* `GET /api/v2/tasks/status/{status}` — фильтрация по статусу (`completed` / `pending`).
* `GET /api/v2/tasks/quadrant/{quadrant}` — фильтрация по квадранту (`Q1`-`Q4`).
* `POST /api/v2/tasks/` — создание новой задачи.
* `PUT /api/v2/tasks/{id}` — полное обновление задачи.
* `PATCH /api/v2/tasks/{id}/complete` — отметка задачи как выполненной.
* `DELETE /api/v2/tasks/{id}` — удаление задачи.

## Запуск проекта
1. Настройте подключение к БД в файле `.env`:
   ```text
   DATABASE_URL=postgresql+asyncpg://user:pass@host:port/dbname
   ```
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Запустите сервер:
   ```bash
   uvicorn main:app --reload
   ```

Документация Swagger доступна по адресу: `/docs`
